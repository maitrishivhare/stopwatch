Stopwatch Web Application
A responsive and interactive Online Stopwatch built using HTML, CSS, and JavaScript.
This project demonstrates the use of JavaScript timing functions and DOM manipulation to create a functional stopwatch web application.

Live Demo
Live Website:
https://maitrishivhare.github.io/stopwatch/
Repository
GitHub Repository:
https://github.com/maitrishivhare/stopwatch

Features
Start the stopwatch
Stop/Pause functionality
Reset timer to zero
Displays minutes, seconds, and milliseconds
Clean and minimal user interface
Responsive design for different screen sizes

Technologies Used
HTML5 – Structure
CSS3 – Styling and layout
JavaScript – Stopwatch logic and time control

Project Structure
stopwatch/
│
├── index.html
├── style.css
└── script.js

How It Works
JavaScript setInterval() is used to track elapsed time.
Event listeners handle Start, Stop, and Reset button actions.
Time updates dynamically in the DOM.
Reset clears the interval and restores all values to zero.

How to Run Locally
Clone the repository:
git clone https://github.com/maitrishivhare/stopwatch.git
Open the project folder.
Open index.html in your browser.
